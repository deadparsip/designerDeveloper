CSS3 animations are a way of animating objects from one CSS style to another. Animations are created with two components: first, a style describing the CSS animation, and second, a keyframe that indicates each stage of the animation's transition.

CSS animations are easy to implement and are less of a drain on the browser's resources than javascript animations.

HTML5 SVG

Scalable Vector Graphics (SVG) define graphics in XML format. SVG graphics do NOT lose any quality if they are zoomed or resized. Every element and every attribute in SVG files can be animated.

The biggest advantage SVG has over Flash is the compliance with other standards (e.g. XSL and the DOM). Flash relies on proprietary technology that is not open source.

As SVG is XML based, every element is available within the SVG DOM, so we can attach JavaScript event handlers for an element.

Note: This example only works in Firefox and Google Chrome.

You can also create shadows, gradients and other effects..
